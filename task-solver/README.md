# Task Solver

Консольное приложение, которое решает одну задачу **четырьмя разными способами**
промптинга и просит GLM оценить, какое решение точнее. На общем ядре
[`../core`](../core).

## Четыре способа

1. **Простой запрос** — задача отправляется модели как есть.
2. **Пошаговое решение** — та же задача с требованием расписать решение по шагам.
3. **Двухшаговый** — сначала GLM составляет промпт под задачу, затем решает её по
   этому промпту.
4. **Панель экспертов** — промпт собирает группу экспертов (их состав задаёт
   пользователь) и просит выработать общее решение.

После этого пятым запросом GLM получает все решения и выбирает наиболее точное с
обоснованием. Итого 6 запросов к API (способ 3 — два запроса).

## Требования

- Node.js **>= 22.6**.
- Каталог `../core` рядом (импортируется напрямую).
- Свой `.env` в этой папке (см. `.env.example`): `LLM_API_KEY`, `LLM_BASE_URL`,
  `LLM_MODEL`.

## Установка и запуск

```bash
npm install
cp .env.example .env   # впишите ключ/URL/модель
npm run solve
```

Приложение интерактивно спросит:

```
Задача: Чему равно 17 × 24?
Эксперты (через запятую): математик, бухгалтер
```

и напечатает все четыре решения, а затем блок `=== Оценка GLM ===` с вердиктом.
Если эксперты не указаны, модель подбирает их сама.
